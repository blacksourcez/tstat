## Local Env

#Python 2.7

#InfluxDB 1.4

For influxDB 8086 port used

#Chronograph 1.4

Instead of 8888 which is default, 8080 port has been used

## Setup

#Start InfluxDB

/etc/rc.d/init.d/influxdb start

#Start Chronograph

/etc/rc.d/init.d/chronograf start

## Run

./run.py
